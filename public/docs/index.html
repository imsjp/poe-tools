Useful Path of Exile Tools

Useful Path of Exile Tools

Path of Exile is a game that rewards efficiency. Spend less time looking at useless loot, confusing skill trees, and complicated item modifiers with these external tools.

Choosing Your Build

The Path of Exile forums provides a great starting point for the type of characters you would like to play.

Path of Building

Path of Building is a tool that allows Path of Exile players to simulate the skill tree and in-game equipment to evaluate builds.

Loot Filters

Loot filters allow you to decrease the amount of items that appear on the ground for you to click. This increases in-game efficiency with less time spent looking at low-value items.

Trade Macros

Trade macros are external tools that allow players to check item values and trade for items with other players more easily.

Exilence Next

Exilence Next is an app that lets you track your income and net worth in Path of Exile based off the items in your stash.
